# 💸 Expense Tracker

A modern and responsive expense management web application built with Vanilla JavaScript that helps users monitor spending habits, manage monthly budgets, and track financial health through an intuitive user interface.

🔗 Live Demo:
https://namanjainb3-tech.github.io/expense-tracker/

---

## 📖 Overview

Expense Tracker is designed to simplify personal finance management by allowing users to record expenses, categorize spending, define monthly budgets, and monitor remaining balances in real time.

The application stores all data locally within the browser, ensuring a fast, lightweight, and privacy-friendly experience without requiring any backend services.

---

## ✨ Key Features

### 📌 Expense Management
- Add new expenses instantly
- Delete unwanted records
- Automatic expense list updates
- Persistent expense history

### 📊 Budget Tracking
- Set monthly spending limits
- Real-time balance calculation
- Remaining budget visualization
- Budget update functionality

### 🗂 Category-Based Analysis
Track expenses across multiple categories:

- Food
- Travel
- Bills
- Shopping
- Others

### 💾 Local Storage Support
- Automatic data persistence
- No account required
- Data remains available after browser refresh

### 🌙 Theme Switching
- Light Mode
- Dark Mode
- Theme preference saved automatically

### ⌨️ Productivity Shortcuts
Designed for faster interaction using keyboard controls.

| Shortcut | Action |
|-----------|----------|
| Ctrl + Enter | Open Add Expense Form |
| Enter | Save Expense |
| Ctrl + B | Set Monthly Budget |
| Esc | Close Form |
