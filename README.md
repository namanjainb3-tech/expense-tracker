# 💸 Expense Tracker

A responsive expense tracking web application built using vanilla JavaScript. It helps users track daily expenses, analyze category-wise spending, manage a monthly budget, and monitor remaining balance — all with persistent local storage.

## 🚀 Features
- Add and delete expenses
- Category-wise expense tracking (Food, Travel, Bills)
- Monthly budget setting with live balance updates
- Persistent data using Local Storage
- Keyboard shortcuts for faster interaction
- Day / Night mode
- Fully responsive design

## ⌨️ Keyboard Shortcuts
- **Ctrl + Enter** → Open add expense form  
- **Enter** → Save expense  
- **Ctrl + B** → Set monthly budget  
- **Esc** → Close expense form  

## 🛠️ Built With
- **HTML5**
- **CSS3** (Flexbox, Media Queries, clamp)
- **Vanilla JavaScript** (DOM manipulation & localStorage)

## 📱 Responsiveness
The application is optimized for mobile, tablet, and desktop screens using flexible layouts, max-width containers, scalable typography, and targeted media queries.
